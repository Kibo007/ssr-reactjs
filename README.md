# SSR React

## How to use

Download the example [or clone the whole project](https://github.com/nenjotsu/ssr-reactjs):

```bash
git clone https://github.com/nenjotsu/ssr-reactjs.git
cd ssr-react
```

Install it and run:

```bash
yarn install
yarn start
```
